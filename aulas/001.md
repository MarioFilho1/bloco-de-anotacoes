# Dicas do Terminal

Vou listar alguns comandos que estamos aprendendo:

- `cd`  (navega entre pastas)

Exemplo de como entrar numa pasta:
```
cd nomedapasta
```
Exemplo - Sair de uma pasta:
```
cd.. ou apenas os ..
```
- `mkdir` (Criar uma pasta)

Exemplo de como criar uma pasta
```
mkdir nomedapasta
```
